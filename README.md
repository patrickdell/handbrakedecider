# HandBrake Configuration Wizard

A single-file, mobile-friendly wizard that recommends HandBrake settings for the
compress → edit in Final Cut Pro → publish workflow. Built around Constant Framerate
(CFR) and Average Bitrate (ABR) encoding with H.264, plus a file size / processing
time estimator and three quick-reference presets.

No build step — open [`index.html`](index.html) directly in a browser, or host it
as a static file anywhere.
