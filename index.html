Adjust Output

Editing uses Constant Quality (CRF) for predictable visual quality; publishing uses Average Bitrate (ABR) for predictable file size.

Hardware encoders (VideoToolbox/NVENC/QSV) use their own preset/quality scale — the speed preset shown below is an x264 approximation, not exact for hardware encoders.

Passthrough keeps the source's original audio codec/channels untouched — size becomes unpredictable, excluded from the estimator below.

CFR Required

Constant Framerate (CFR) is mandatory. In HandBrake's Video tab, set Framerate (FPS) to your source value and select Constant Framerate — not Variable or Peak. Variable framerate causes audio drift once the file hits Final Cut Pro's timeline.
